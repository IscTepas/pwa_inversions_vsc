## ¿Qué es este repositorio?

Este repositorio implementa la **Metodología Híbrida AI SKILL DEVELOPMENT + SPEC DRIVEN ASSISTANCE**
para gestionar proyectos de software de manera profesional, modular, reutilizable y trazable,
con Inteligencia Artificial como asistente activo en cada fase del desarrollo.

La metodología permite trabajar con un equipo de **agentes IA especializados** que colaboran
en distintas etapas: análisis, desarrollo, optimización y testing.

## Cómo usar este repositorio

### Primera vez (FASE 0 — ya hecha)
La estructura de carpetas, templates y metodología ya están colocados.

### Iniciar con la IA (en orden)
1. Abre tu herramienta de IA (Claude, Copilot, etc.)
2. Usa el prompt: `ai_global/prompts/001_methodology_context_ai.md`
3. Espera confirmación de la IA
4. Usa el prompt: `ai_global/prompts/002_begin_phase_1_ai.md`
5. Autoriza los cambios que la IA proponga
6. Usa el prompt: `ai_global/prompts/003_begin_phase_2_ai.md`
